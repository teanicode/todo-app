# Todo App

A simple Python command-line todo application with persistent file storage.

## Features

- Add new tasks
- View all tasks
- Mark tasks as complete
- Save tasks to file (tasks persist after closing app)

## How to Use

1. Run the app:

2. Follow the menu:
   - Enter 1 to add a task
   - Enter 2 to view tasks
   - Enter 3 to mark a task complete
   - Enter 4 to save and exit

## Files

- `main.py` - Main application code
- `tasks.json` - Saved tasks (created after first save)

## Technologies

- Python 3
- JSON for file storage
